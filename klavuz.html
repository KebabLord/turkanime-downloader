TürkAnimu İndirici - Kılavuz

Bu scriptin veya kodun veya programın, her ne derseniz, başlıca özellikleri:

  • Yığın ve tekil anime indirme otomasyonu
  • Anime izleme ve izlerken kaydetme
  • Uygulama içinden arama yapmak ve çözünürlük seçebilmek

Kurulum

Windows
Mpv ve Geckodriver'ın son sürümünü indirip program dizinine atın ya da bulundukları dizini sistem PATH'ına ekleyin. Ardından gerekli paketleri kurun: py -m pip install -r requirements.txt yada pip3 install -r requiremenets.txt. Uygulama bir de youtube-dl'e ihtiyaç duyuyor, youtube-dl'i indirebilir yada indirmenize gerek kalmadan python dizinindeki scripts klasörünü sistem PATH'ınıza cmd'de şu şekilde ekleyebilirsiniz: set PATH=%PATH%;C:\PYTHON'UN_DİZİNİ\Scripts

MacOS
Brew paket yöneticisini yükledikten sonra brew install geckodriver mpv komutuyla gerekli uygulamaları kurun. Ardından pip3 install -r requirements.txt ile gerekli paketleri kurun.

Linux
Paket yöneticinizle veya github sayfalarından mpv ve geckodriver'ı kurduktan sonra gerekli paketleri pip3 install -r requirements.txt ile kurun.

İzlerken kaydetme modu

Mpv’nin deneysel bir modülüdür. Ayarlardan aktifleştirip devre dışı bırakabilirsiniz.
